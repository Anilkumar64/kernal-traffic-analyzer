```

```

## TECH DECISIONS MADE

- **No QtWebEngine** — broken on Ubuntu 22.04 (6.2.4+dfsg-6ubuntu1), missing binary
- **Map approach** — use QNetworkAccessManager + QGraphicsScene tile rendering instead
- **Font** — Ubuntu Mono (confirmed installed: /usr/share/fonts/truetype/ubuntu/UbuntuMono-B.ttf)

- **No Electron/React** — pure Qt6 desktop, Anil is C++ developer
- **Dijkstra not applicable** — we're visualizing the ACTUAL path packets took (from traceroute), not computing shortest path. Haversine formula for geographic distances between hops.

## KERNEL CODING RULES (for reference)
1. DEFINE_HASHTABLE never with static keyword
2. Stack frames under 1024 bytes
3. get_mm_exe_file() removed in 6.7 — use rcu_dereference(mm->exe_file)
4. spin_lock_bh() not spin_lock() in softirq context
5. GFP_ATOMIC for all kernel allocations in softirq context
6. Always put_pid() after find_get_pid()
